## Featured Projects / 代表项目

### [YOLOv11 UAV Small-Object Detection](https://github.com/OOOHGOD/YOLOv11-Based-Small-Object-Detection-System-Demo-for-UAV-Aerial-Images)

基于 YOLOv11 的无人机航拍小目标检测系统，融合 PPA 注意力模块与 SAHI 切片推理，提供图片、视频、摄像头和批量检测的 PyQt5 可视化界面。

### [Multi-feature Fundus Disease Recognition](https://github.com/OOOHGOD/Resnet_Eye_Class_ODIR)

面向 ODIR-5K 双目眼底彩照的多标签识别系统；采用共享参数的双分支 ResNet18d、特征融合、Focal Loss 与动态分类阈值，并配套 Web 端诊断流程。

`Python` `PyTorch` `ResNet` `Computer Vision` `Vue.js` `MySQL`

### [YOLOv8 X-ray DangerScan](https://github.com/OOOHGOD/YOLOv8_X_ray_DangerScan)

覆盖标签格式转换、YOLOv8 训练验证、模型导出与 PyQt5 推理界面的 X-ray 危险品检测项目，支持 8 类常见物品识别。

`Python` `YOLOv8` `OpenCV` `PyQt5` `ONNX` `TensorRT`
